# StudyMate

## Project Overview
StudyMate is a web application designed to help users efficiently track their study habits. 

## Features
- Upload study data
- Visualize progress
- Share insights with peers

## Technologies Used
- HTML
- CSS (Bootstrap)
- JavaScript
- Flask
- SQLAlchemy
- Chart.js
